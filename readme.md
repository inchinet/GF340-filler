# 🚀 Antigravity | GF340 自動填表工具

一個以瀏覽器執行的工具，可幫手自動填寫香港政府 **G.F. 340** 網上申請表格。
本工具支援 **中、英文雙語介面**，會根據你開啟的政府頁面版本自動調整填表邏輯。
所有資料均於**本機處理**，不會上傳至任何伺服器。

> **僅供個人使用。** 本工具由 [inchinet](https://github.com/inchinet) 為個人用途開發。
> 請勿公開分享已填寫的資料檔案。

## 📋 資料欄位參考
 
你可以選擇使用**繁體中文**或**英文**欄位名稱（English tags）。
- **中文版**：編輯 `sample_data.txt`
- **英文版**：編輯 `sample_data_en.txt`

欄位名稱**區分大小寫**，以 `: ` 分隔欄位名稱與數值。
以 `#` 開頭的行為注釋，系統會自動略過。

> **提示：** 如某行不需填寫（例如第7筆工作記錄），請將數值留空：
> ```
> 全職7:
> 機構7:
> 職位7:
> 工作性質7:
> 由7:
> 至7:
> ```
> 空白數值會被系統自動略過，**不會**在政府網站上新增多餘的行。

## 🚀 使用步驟

1. **建議瀏覽器** — 建議使用 **Google Chrome** 或 **Microsoft Edge**。

2. **開啟 GF340 網上申請頁面** — 進入
   [JOA_005.action](https://csboa1.csb.gov.hk/csboa/joa/JOA_005.action)
   （即所有文字輸入欄位所在的頁面）。**支援中文或英文版面。**

3. **準備資料** — 根據你的喜好複製 `sample_data.txt` (中文) 或 `sample_data_en.txt` (英文)，用記事本填入你的個人資料。

4. **開啟填表工具** — 在瀏覽器中開啟 `filler_dashboard.html`。
   - 將你的資料內容貼入**左側文字區域**。
   - 點擊 **「Generate & Copy Script」**。

5. **將腳本注入表格**：
   - 切換回 **GF340 網頁**。
   - 按 **`F12`** → 點擊 **Console（主控台）** 分頁。
   - 貼上（`Ctrl+V`）並按 **Enter**。
   - ⚠️ 若出現安全性警告，請先輸入 `allow pasting` 並按 Enter 即可。

6. **等待完成** — 腳本會自動新增行數並填寫。
   - **重要：** 因應政府網站動態載入需求，填寫考試科目時會有約 0.6 秒的延遲等待，請耐心等候直至彈出 `Transfer complete (Bi-lingual Mode)!`。

7. **提交前請核對** — 腳本不會自動提交。請檢查無誤後自行按「儲存並繼續」。

---

## ⚠️ 注意事項

- **擴充功能衝突**：請暫時關閉 **uBlock Origin**、**AdBlock** 或 **Dark Reader** 等瀏覽器擴充功能。這些工具可能會封鎖政府網站的 JavaScript 或干擾腳本注入，導致填表失敗。
- **先重新整理頁面**：執行前請先按 `F5` 確保狀態乾淨。

- **日期格式**：
  - 工作記錄 → `DD/MM/YYYY`
  - 考試／學歷 → `MM/YYYY`
- **會話逾時**：政府網站可能會自動登出，重新執行腳本即可。
