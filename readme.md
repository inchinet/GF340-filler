# 🚀 Antigravity | GF340 自動填表工具

一個以瀏覽器執行的工具，可幫手自動填寫香港政府 **G.F. 340** 網上申請表格，「智方便」並未能填所有履歷資料。
所有資料均於**本機處理**，不會上傳至任何伺服器。

> **僅供個人使用。** 本工具由 [inchinet](https://github.com/inchinet) 為個人用途開發。
> 請勿公開分享已填寫的資料檔案。

## 📋 資料欄位參考

請按照以下**繁體中文欄位名稱**編輯 `sample_data.txt`。
欄位名稱**區分大小寫**，以 `: ` 分隔欄位名稱與數值。
以 `#` 開頭的行為注釋，系統會自動略過。

> **提示：** 如某行不需填寫（例如第7筆工作記錄），請將數值留空：
> ```
> 全職7:
> 機構7:
> 職位7:
> 工作性質7:
> 由7:
> 至7:
> ```
> 空白數值會被系統自動略過，**不會**在政府網站上新增多餘的行。

## 🚀 使用步驟

1. **開啟 GF340 網上申請頁面** —  直至進入
   [JOA_005.action](https://csboa1.csb.gov.hk/csboa/joa/JOA_005.action)
   （即所有文字輸入欄位所在的頁面）/ 使用英文版便可。

2. **準備資料** — 複製 `sample_data.txt`，填入你的個人資料。

3. **開啟填表工具** — 直接在瀏覽器中開啟 `filler_dashboard.html`（無需伺服器）。
   - 將資料檔案的內容貼入**左側文字區域**。
   - 右側面板會顯示偵測到的考試／學歷／工作行數。
   - 點擊 **「Generate & Copy Script」**。

4. **將腳本注入表格**：
   - 切換回 **GF340 瀏覽器分頁**。
   - 按 **`F12`** → 點擊 **Console（主控台）** 分頁。
   - 貼上（`Ctrl+V`）並按 **Enter**。
   - ⚠️ 如 Chrome 顯示警告，請先輸入 `allow pasting` 並按 Enter，然後再重新貼上。

5. **等待完成** — 腳本會自動新增所需行數並填寫所有欄位。
   8 行考試 + 8 行工作記錄約需 15–20 秒。完成後會彈出提示訊息(AntiGravity: Transfer complete!)

6. **提交前請核對** — 請仔細檢查所有欄位。腳本不會自動提交表格。

---

## ⚠️ 注意事項

- **先重新整理頁面**：執行腳本前，請先按 `F5` 重新載入 GF340 頁面，確保狀態乾淨。
- **地區下拉式選單**：請以英文輸入地區名稱（例如 `Yuen Long`），腳本會自動翻譯成中文。
- **日期格式**：
  - 工作記錄日期 → `DD/MM/YYYY`
  - 考試／學歷日期 → `MM/YYYY`
- **會話逾時**：政府網站可能會自動登出。如遇此情況，重新登入後再執行腳本即可。
- **空白行的處理**：不需填寫的行，將數值留空即可（例如 `機構3:` 冒號後不填任何內容）。
  空白數值會被自動略過，**不會**觸發多餘的新增行動作。
